## Install

### 1. Clone the source code

```shell
git clone git@github.com:KlimovDmitry/yii-url-shortener.git
```

### 2. Install the extended package dependency.

Install the `yii2-app-basic` repositories: 

```shell
composer install
```

### 3. Running Migrations.

```shell
yii migrate
```
